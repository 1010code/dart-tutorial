# 數字類型 (Numbers)

- 數字類型：整數 int、浮點數 double

## 整數 int
宣告一個整數變數，並賦予初值。

```dart
int num1 = 10; 
```


## 浮點數 double
64-bit 雙精度浮點數。

```dart
double num2 = 12.5; 
```

也可以將變數宣告為 num 類型，系統會認定允許 int 和 double 數值。

```dart
num x = 1; // x 可以是整數或浮點數
x += 2.5;
```

另外我們可以透過 `.runtimeType` 查詢該變數的型態。

```dart
num x = 1;
print(x.runtimeType);
x += 2.5;
print(x.runtimeType);
```

輸出結果：
```
int
double
```
