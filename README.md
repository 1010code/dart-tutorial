## DartPad
初學者可以專心在程式學習上，非常建議使用 [DartPad](https://dartpad.dartlang.org/) 線上編譯器進行 Dart 程式語言練習。其優點是無需安裝任何軟體，只要可以使用瀏覽器上網即可以立即使用。
